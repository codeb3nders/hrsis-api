## Installation

# Download and install MongoDB on your machine.

# (download link) https://www.mongodb.com/docs/manual/administration/install-community/

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Swagger
# Api Documentation link -> http://localhost:3000/swagger/
